
# Amason

Amason est une application de commerce électronique développée en utilisant le framework JavaScript React. L'objectif principal de cette application est de permettre aux utilisateurs de parcourir une sélection de produits, de les ajouter à leur panier et de passer des commandes.

## Fonctionnalités

- Parcourir une gamme de produits disponibles
- Rechercher des produits par catégorie ou par mot-clé
- Ajouter des produits au panier
- Gérer le panier en ajoutant, supprimant ou mettant à jour les quantités des produits
- Passer des commandes avec un processus de paiement simplifié

## Installation

Pour installer et exécuter localement ce projet, suivez ces étapes :

1. Clonez ce dépôt sur votre machine locale en utilisant la commande suivante :

   ```
   git clone https://github.com/Paul-FranckDencausse/Amason.git
   ```

2. Accédez au répertoire du projet :

   ```
   cd Amason
   ```

3. Installez les dépendances en exécutant la commande :

   ```
   npm install
   ```

4. Lancez l'application en utilisant la commande :

   ```
   npm start
   ```

5. Ouvrez votre navigateur et accédez à l'adresse [http://localhost:3000](http://localhost:3000) pour voir l'application en cours d'exécution.

## Technologies Utilisées

- React.js
- HTML5
- CSS3
- JavaScript (ES6+)
- Firebase (pour la gestion des utilisateurs et des commandes)
